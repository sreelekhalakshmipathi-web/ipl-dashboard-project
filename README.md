# ipl-dashboard-project
IPL data analysis using python
This project analyzes IPL (Indian Premier League) data using Python and visualizes key insights through a dashboard.

## 📊 Features
- Top winning teams analysis
- Top batters performance
- Toss decision trends
- Matches per season trends

## 🛠 Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📌 Key Insights
- Mumbai Indians have the highest number of wins
- Teams often prefer chasing after winning the toss
- Player performance varies across seasons

## 🚀 How to Run
1. Download the notebook
2. Run in Jupyter Notebook / Kaggle
